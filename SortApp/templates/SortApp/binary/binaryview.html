What is Binary Search ?

   Binary Search is a technique to search an element in the list using the divide and conquer technique. It is a very fast and efficient searching technique. This type of technique is used in the case of sorted lists. In this method, to search an element you might compare that respective element present in the center of the list and if it’s same then the search is successfully finished and if not, then the list is divided into two parts: one from 0th element to the centered element and other from the centered element to the last element.

   The time and space complexity for binary search is:
• Time complexity (Best Case) : O(1)
• Time complexity (Worst Case) : O(log n)
• Time complexity (Average Case) : O(log n)
• Space complexity : O(1)

Algorithm :

Binary Search Algorithm is as follows:
Step 1: Set BEG = LB, END = UB and MID = INT([BEG+END]/2)
Step 2: Repeat step 3 and 4 while BEG <= END and ARR[MID] !=    ITEM
Step 3: IF ITEM< ARR[MID] then: Set END = MID-1
   Else:
   Set BEG = MID+1
Step 4: Set MID = INT(BEG+END)/2
Step 5: IF ARR[MID] = ITEM then: Set LOC = MID
   Else:
   Set LOC = NULL
Step 6: Exit

Implementation of Binary Search :


Step 1 - In the Given array, first you need to find the middle of the list using formula mid = (min + max)/2.
Step 2 - If the searched element that is; say 50 is in the centered then the search is terminated, as we can see here it is not.
Step 3 - With the help of this algorithm we will try to find the value. First of all, you have to define two variables, low and high so low is 10 and high is 80. Now you have to find the mid element.so we will find the mid value of the formula.
Step 4 - We would know that in which half the searching would be done.
Step 5 - So we would go for the second half as the searching element that is; 60 is greater than the mid element that is 50.
Step 6 - The process repeat on until we find the element.
