What is Linear Search ?

Linear Search is also called as sequential search or serial search. This is the traditional technique for searching an element in a collection of elements. In this method of searching, the element to be found is searched sequentially in the list. It begins to search first item in list, continues searching sequentially (item by item) through the list, until desired item (key) is found or until end of list is reached. If the element is not found till the end this means that the element is not present in the list. This method can be performed on a sorted or an unsorted list.

The time and space complexity for linear search is: • Time complexity (Best Case) : O(1) • Time complexity (Worst Case) : O(n) • Time complexity (Average Case) : O(n) • Space complexity : O(1)

Algorithm

Linear Search Algorithm is as follows: Step 1: Set i to 1 Step 2: If i > n then go to step 7 Step 3: If A[i] = x then go to step 6 Step 4: Set i to i + 1 Step 5: Go to Step 2 Step 6: Print Element x Found at index i and go to step 8 Step 7: Print element not found Step 8: Exit

Implementation of Linear Search:

Step 1 - The list given is the list of elements in an unsorted array. Step 2 - The element to be searched is called item, as mentioned in the image, so value item, is compared with all the elements of the array starting from 0th element. Step 3 - As soon as any index contains the same value, we stop further comparisons and change flag value from 0 to 1. Step 4 - If the value is not found until the end of the array then stop and print value not found.
