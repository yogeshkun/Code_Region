What is Selection Sort ?

   Selection sort is a sorting algorithm that selects the smallest element from an unsorted list in each iteration and places that element at the beginning of the unsorted list. It is one of the slow sorting technique. Given an unsorted array of integer values, a selection sort visits each element of the array, in turn. As it visits an element, it employs a second index to scan the array from the present location to the end while it identifies the least (smallest) value in that segment of the array. It then swaps that least value with the current value.

   The time and space complexity for bubble sort is:
• Time complexity (Best Case) : Ω(n)
• Time complexity (Worst Case) : O(n2)
• Time complexity (Average Case) : θ(n2)
• Space complexity : O(1)

Algorithm :

Selection sort Algorithm is as follows:
• SELECTION SORT(ARR, N)
Step 1: Repeat Steps 2 and 3 for K = 1 to N-1
Step 2: CALL SMALLEST(ARR, K, N, POS)
Step 3: SWAP A[K] with ARR[POS]
   [END OF LOOP]
Step 4: Exit

• SMALLEST (ARR, K, N, POS)
Step 1: [INITIALIZE] SET SMALL = ARR[K]
Step 2: [INITIALIZE] SET POS = K
Step 3: Repeat for J = K+1 to N -1
   IF SMALL > ARR[J]
   SET SMALL = ARR[J]
   SET POS = J
   [END OF IF]
   [END OF LOOP]
Step 4: Repeat for J = K+1 to N -1

Example demonstrating Selection sort:

Suppose we have the following array, which we have to sort.
22 18 16 10 19 15
There are 6 elements in this array, so we will need 5 steps to sort it.

Step 1:-
The first position in the array has 22 stores.
If you search in the whole array, you get the smallest value 10, and then you replace 22 with 1.
10 18 16 22 19 15

Step 2:-
The second position in the array has 18 stores.
If you search in the whole array, you get the second smallest value 15, and then you replace 18 with 15.
10 15 16 22 19 18

Step 3:-
The third position in the array has 16 stores, but there will be no change in this place because there is no element smaller than 16, which means that it is already sorted.
10 15 16 22 19 18

Step 4:-
The fourth position in the array has 22 stores.
If you search in the whole array, you get the second smallest value 18, and then you replace 22 with 18.
10 15 16 18 19 22

Step 5:-
The fifth position in the array has 19 stores, but there will be no change in this place because there is no element smaller than 19, which means that it is already sorted.
10 15 16 18 19 22
